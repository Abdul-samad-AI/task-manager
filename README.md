# Task Manager (Full Stack)

## Description
Task Manager is a full-stack web application designed to help users manage their daily tasks efficiently.
Users will be able to create, update, delete, and track tasks using a clean and user-friendly interface.

This project is being built to practice real-world full-stack development using React and Django.

---

## Tech Stack
- Frontend: React
- Backend: Django, Django REST Framework
- Database: PostgreSQL (planned)
- Authentication: JWT (planned)

---

## Planned Features
- User authentication (login & signup)
- Create, update, delete tasks
- Mark tasks as completed or pending
- User-specific task dashboard
- Secure API-based communication
